# 小红书 AI 内容运营工作流

这是给运营使用的低代码内容生产画布。运营输入需求、竞品链接、评论线索、指定选题和 API Key 后，工作流会完成竞品研究、爆款选题、图片技能展开、文案/套图生成、成品审核和复盘建议。

## 运行

```bash
bash scripts/local_run.sh -m flow
bash scripts/local_run.sh -m node -n node_name
bash scripts/http_run.sh -m http -p 5000
```

运行前必须由用户输入：

- `openai_api_key`
- `grok_api_key`

## 运营工作流

```text
运营需求入口
└─ AI理解运营目标
   ├─ 竞品和用户需求分析
   ├─ 爆款选题机会池
   ├─ 图片制作技能子树
   ├─ 文案技能子Agent
   ├─ 图片技能子Agent
   ├─ 运营确认生成方案
   ├─ AI生成小红书文案
   ├─ AI生成小红书套图
   └─ 成品审核与复盘建议
```

## 选题规则

- 有 `user_selected_topic` 时，运营指定选题优先。
- 有 `xiaohongshu_url` 且真实运行时，OpenAI 会联网查询竞品链接并给出选题灵感。
- 话题标签来自运营指定选题或 OpenAI 选题结果，最终审核节点不再写死标签。

## 运营如何改技能

运营看到的是可展开的技能流程，不需要看代码：

- 图片制作技能：风格、比例、必备元素、避坑清单、套图统一。
- 文案技能：理解选题和用户痛点，生成可发布的图文文案。
- 图片技能：确定套图画面方向，生成每页图片。
- AI技能教练：判断改动是本次临时使用，还是值得沉淀为长期技能经验。
